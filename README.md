Sales Performance Dashboard (Power BI)
Project Overview:-
This project presents an interactive sales analysis dashboard built using Power BI.
It analyzes regional performance, category trends, profitability, and monthly sales behavior.

Tools Used:-
- Power BI
- DAX
- Excel

Features:-
- KPI cards (Sales, Profit, Margin)
- Region & Category analysis
- Monthly trend charts
- Scatter profitability analysis
- Interactive slicers

Insights:-
- Identified top performing regions
- Detected low margin products
- Observed seasonal sales trends

Files Included:-
- PowerBI_Report.pbix
- Dataset.xlsx
- Dashboard Screenshot

This project demonstrates data cleaning, modeling, visualization, and business analysis skills.
